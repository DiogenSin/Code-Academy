Formos elementai Formos atributai Input tipai
  • form - forma

  • button - mygtukas. Jie turi turėti rolės atributą (pvz. role="submit")

  • name - nurodo įvesties lauko pavadinimą, kuris bus perduotas back-end`ui kartu su įvesties reikšme.

  • cols - textarea elemento plotis stulpeliais.

  • value - įvesties reikšmė perduodama į serverį.

  • password - slaptažodis (paslepia kas rašoma).
