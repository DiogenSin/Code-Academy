Tag`ų sąrašas

  • p - paragraph - paragrafas. Skirta rašyti didesniam kiekiui tekstinės informacijos.
  • h1 - h6 - heading - antraštės skirtos rašyti pavadinimams ir pan.
  • img - image - paveiksliukas - skirtas įkelti paveiksliuką, naudojami src, alt, width, height atributai
  • a - anchor - nuoroda, nukelianti vartotoją į norodytą lokaciją. Naudojamas atributas href.)
  • br - (line) break - eilutės lūžis
  • hr - horizontal break - eilutės lūžis su horizontalia linija.
  • ul - unordered list - nerikiuotas sąrašas.
  • ol - ordered list - rikiuotas sąrašas.
  • li - list item - sąrašo elementas skirtas ol arba ul sąrašams
  • meta - skirtas nurodyti puslapio meta duomenis.
  • title - skirtas nurodyti puslapio pavadinimą.

Atributų sąrašas

  • lang - language - kalba, vyraujanti puslapyje. Naudinga automatiniams skaitytuvams.
  • charset - Character set - simbolių rinkinys nurodantis naudotiną simbolių biblioteką (dažniausiai UTF-8)
  • src - source - šaltinis, nurodantis failą.
  • alt - alternatyvusis tekstas - tekstas, apibūdinantis failo turinį ir skirtas automatiniams skaitytuvams.
  • width, height - plotis, aukštis - naudojama vaizdinių elementų atvaizdavime.
  • href - hyper link reference - nuoroda į tam tikra lokaciją arba failą.
  • target - nurodo kaip turi būti atidaroma nuoroda (tame pačiame, naujame ir pan.)

Klaviatūros trumpinių sąrašas

  • ctrl s - išsaugo failą.
  • ctrl c - kopijuoja pažymėtą informaciją.
  • ctrl x - iškerpa pažymėtą informaciją.
  • ctrl v - įklijuoja iškirptą/kopijuotą informaciją.
  • ctrl z - atšaukia paskutinį veiksmą.
  • ctrl y - grąžina atšauktą veiksmą.
  • ctrl / - įjungia komentavimo funkciją.
  • ctrl a - pažymi visą turinį.
  • ctrl f - įjungia tekstinio turinio paiešką.
  • ctrl shift i/j | F12 - atidaro developer tools langą.
  • ctrl shift l - pažymi visus vienodus žodžius/sakinius.
  • ctrl shif k - ištrina eilutę.
  • alt z - word wrap
  • ctrl d - pažymi pažymėtą tekstą žemiau esančioje eilutėje/eilutėse.
  • end - nukels į eilutės galą.
  • ctrl end - nukels į kodo galą.
  • home - nukels kursorių į eilutės pradžią.
  • ctrl home - nukels į kodo pradžią.
  • shift alt (į apačią) - nukopijuos pažymėtą kodą į apačią.
  • shift alt (į viršų) - nukopijuos pažymėtą kodą į viršų.
  • ! - Parašo HTML failo šabloninį turinį.
  • ctrl shift enter - sukuria naują eilutę nelaužiant žodžio pusiau.
  • ctrl shift j> - naršyklėje parenka inspect element įrankį.
  • alt tab - keisti atidarytus langus.
  • windows tab - atidaro desktop`ų kūrimo/pasirinkimo langą.
  • ctrl t - sukuria naują tab`ą.
  • ctrl shift t - atidaro vėliausiai uždarytą tab`ą.
