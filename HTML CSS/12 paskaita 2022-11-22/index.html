Git`as ir GitHub`as

Git`as yra kodo versijavimo programa, o GitHub`as yra kodo saugojimo ir dalijimosi platforma.

  1. Įsirašyti Git`ą
  2. Patikrinti ar jis įrašytas terminale įrašant "git --version"
  3. Sukurti gitHub paskyrą
  4. Sukonfiguruoti git`ą:
    • git config --global user.name DiogenSin
    • git config --global user.email jakasdovydas@gmail.com
    • git config --global core.editor code
    • git config --global core.autocrlf true / input | True jei Windows ir input jei Mac.
    • git config --global -e | skirta patikrinimui.
  5. Patikrinti ar tvarkingas config dailas naudojantis "git config --global --edit"
  6. Sujungti Visual Code su GitHub paskyra Source Control skyrelyje.
  7. Commit ir Publish pakeitimus
