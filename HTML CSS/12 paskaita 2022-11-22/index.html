Git`as ir GitHub`as

Git`as yra kodo versijavimo programa, o GitHub`as yra kodo saugojimo ir dalijimosi platforma.

  1. Įsirašyti Git`ą
  2. Patikrinti ar jis įrašytas terminale įrašant "git --version"
  3. Sukurti gitHub paskyrą
  4. Sukonfiguruoti git`ą:
    • git config --global user.name DiogenSin
    • git config --global user.email jakasdovydas@gmail.com
    • git config --global core.editor code
    • git config --global core.autocrlf true / input | True jei Windows ir input jei Mac.
    • git config --global -e | skirta patikrinimui.
  5. Patikrinti ar tvarkingas config dailas naudojantis "git config --global --edit"
  6. Sujungti Visual Code su GitHub paskyra Source Control skyrelyje.
  7. Commit ir Publish pakeitimus

  • Suvokiamumas - suprantamas ir aiškus turinys.

SASS / SCSS

Very very good

Suteikia galimybę rašyti CSS naujoviškesniu būdu, kuriame galima nesunkiai kurti ir naudoti kintamuosius, tinklinti (nest`inti), rašyti sąlygas, ciklus, funkcijas, mix-in`us ir t.t.
Naudojant kažkurį CSS preprocessor`ių reikia parašytą kodą konvertuoti į įprastą CSS`ą, kurį supranta naršyklė.

Size units

  • rem - 16 px. Valdomas keičiant :root nustatymus.
  • em - valdomas keičiant tėvinio elemento nustatymus.

CSS frameworks

CSS Framework - kažkieno kito parašytas CSS kodas, kurį galima naudoti savo projektui. Norint naudoti kažkieno kito framework`ą, jį reikia prijungti naudojant URL`ą.
Dauguma framework`ų veikia naudojant jų parašytas klases HTML elementams.
Yra Classless CSS framework`ų. Juos naudojant klasės nėra suteikiamos klasių elementams, tačiau viskas aprašyta naudojant tag`ų selektorius.

  • Tailwind
  • Bootstrap
  • Bulma
  • UI Kit
  • Materialize
  • Milligram
  • Foundation
  • ...

Pros

  • Taupomas laikas - 99% CSS kodo jau paruošta naudojimui.
  • Pagreitina darbą
  • Lengva naudoti - Bootstrap atveju pakanka priskirti klases atitinkamiems html elementams.
  • Paruoštas svetainės responsiveness.
  • Puikus sprendimas pradiniam svetainės maketui kurti.

Cons

  • Daug perteklinio/nepanaudoto kodo.
  • Neunikalus dizainas.
  • Reikalauja laiko išmokstant naudotis.
  • Prastesnis Google paieškos sistemos vertinimas.
