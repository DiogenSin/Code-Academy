Mobile first

  • Lengvesnis perėjimas iš mažesnio ekrano dydžio į didesnį.
  • Koncentracija į turinį (dėl mažo vietos kiekio).
  • Didelis mobiliųjų aplikacijų vartotojų skaičius.
  • Google paieškos algoritmas geriau įvertina Mobile First dizainu kurtas svetaines.
  • Naudojant Mobile First dizaino principą, pradedame nuo mažiausio ekrano ir judame link didesnių.
    Tam naudojame @media (min-width: ____px).
    Naudojant @media (min-width: ____px) judame nuo mažesnių dydžių į didesnius.
