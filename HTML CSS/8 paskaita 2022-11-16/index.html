Transformacija

  • scale - keičia elemento dydį (default 1, min 0).
  • scale (x,y), scaleX, scaleY, scaleZ, scale3d (x,y,z)
  • rotate - pasuka elementą (default 1, min 0).
  • rotateX, rotateY, rotateZ, rotate3d (X, Y, Z, A)
  • translate - elemento pozicijos pakeitimas.
  • translateX, translateY, translateZ, translate3d (X, Y, Z)
  • skew - tampo elementą už kampų.
  • skewX, skewY, skew
  • matrix, matrix3D
  • perspective
Labas

Tranzicija

  • transition-property - kokie stiliai keičiasi
  • transition-duration - per kiek laiko įvyksta pakitimai
  • transition-timing-function - tranzicijos laiko paskirstymas
    • ease
    • ease-in
    • ease-out
    • ease-in-out
    • linear
    • cubiz-bezier
  • transition-delay - po kiek laiko pradeda vykti pakitimai

Animacija

  • animation-duration - animacijai skirtas laiko tarpas
  • animation-timing-function - animacijos laiko paskirstymas
  • animation-delay - po kiek laiko prasidės animacija
  • animation-iteration-count - kiek sykių vyks animacija
  • animation-direction - animacijos kryptis (pradžia-pabaiga/pabaiga-pradžia)
  • animation-fill-mode - animacijos pradžios/pabaigos nustatymai
  • animation-play-state - animacijos būsena
  • animation-name - animacijos pavadinimas
  • animation - visų 8 nustatymų junginys

Display: flex

  • flex-direction - nusako, kaip išsidėstys elementai.
  • flex-wrap - perkelia elementus netelpančius į wrap`o rėmą.
  • flex-flow - flex-direction ir flex-wrap junginys (du nustatymai viename).
  • justify-content - nusako, kaip elementai išsidėstys flex`o main`o dalyje.
  • align-items - nusako, kaip atskiri elementai išsidėstys flex`o cross`o dalyje.
  • align-content - nusako, kaip visi elementai kartu išsidėstys flex`o cross`o dalyje.
  • gap/row-gap/column-gap - atitinkami tarpai tarp elementų.
  • order - elementų eilės tvarka.
  • flex-grow - pakeičia elemento plotį.
  • align-self - pakeičia elemento poziciją naudoti su "align-items".
