CSS rašymo būdai

Inline - Linijinis

Linijinis stilius yra, kai elementui, kurį stilizuojame, suteikiame style atributą ir jo viduje aprašome/nurodome stilius. Neturi perpanaudojimo galimybės ir yra skirtas tik testavimui.

Internal - Vidinis

Vidiniu stiliumi laikomas toks stilius, kai HTML failo viduje (head tag`o dalyje, style tag`e) parašomas CSS kodas. Rekomenduotina tik testavimui, tačiau kartais naudojama ir kaip pagrindinis stilizavimo būdas. Didesnis panaudojamumas dėl esančių selektorių. Tinka paprastiems HTML puslapiams iki 100 eilučių.

External - Išorinis

Išorinis stilius yra rašomas atskirame CSS faile. Sukuriamas main.css failas ir prijungiamas prie HTML failo naudojantis link css funkcija. Turi visas perpanaudojimo galimybes ir dažniausiai naudojamas svetainių kūrimo projektuose. Galutinio projekto faile turėtų likti tik css faile esantis stilizavimas.

Basic Selectors

Element / Tag selector

Select`inami visi nurodyto tag`o elementai.
Pvz.: h2 -> parinks visus h2 elementus HTML`e.

Class selector

Klasės pradedamos rašyti tašku ".". Klasės pavadinimas (idealiu atveju) yra trumpas, logiškas ir suprantamas. Klasė nurodoma CSS faile.
Pvz.: .svarbustekstas{}.

Norint klasę priskirti elementui HTML faile yra naudojamas nurodomas atributas "class" su atitinkama reikšme.
Pvz.: class="svarbustekstas

Šitam tekstui priskirta klasė "svarbustekstas".

ID selector

ID sukuriamas bet kokiu norimu logišku ir trumpu pavadinimu. CSS`e ID nurodomas grotelėmis (#). Didžiosios ir mažosios raidės turi reikšmės.
Pvz.: #idPavadinimas{}

HTML`e ID priskiriamas naudojant atributą ID ir jo pavadinimą.
Pvz.: id="idPavadinimas"

Šitam tekstui priskirta ID klasė.

Skirtumas tarp Class ir ID yra tas, jog viena klasė gali būti priskirta daugeliui elementų ir tuo pačiu daug klasių gali būti priskirta vienam elementui. Šiam tekstui priskirtos dvi klasės.
ID turi vienas-su-vienu sanktykį. Vienas elementas gali turėti tik vieną ID ir vienas ID turėtų būti priskirtas tik vienam elementui.

Colors

Spalvoms naudojamas HEX, RGB arba vardų kodai.
Spalvų vardų gali būti ~140. Norint elementui priskirti spalvos nustatymą, reikia naudoti spalvos vardą/kodą.
Pvz.: white, black, lightgrey, darkgreen ir t.t.

RGB - Red Green Blue

RGB - red green blue. Varijuojant kiekvieną iš šių trijų spalvų (pagal kiekį) nuo 0 iki 255 išgaunamos įvairios spalvos.
Pvz.: rgb(0, 0, 0) - juoda, rgb(255, 0, 0) - raudona.

RGBA - red green blue alpha. Alpha nurodo spalvos permatomumą. Permatomumo lygis parenkamas nuo 0 iki 1 su tūkstantąja paklaida.
rgba (0, 255, 0, 0) - nesimato. 0 apha = visiškai persimato, 1 apha = nepersimato.

HEX - Hexadecimal

HEX - Hexadecimal - šešioliktainis. Šešioliktainė sistema susideda iš 0-9 ir a-f. HEX yra tas pats kas RGB, tik nusakoma ne trimis spalvomis, o kombinuojant vertę nuo 00 iki ff.
Skaičiavimas HEX sistemoje: 00 01 02 03 04 05 06 07 08 09 0a 0b 0c 0d 0e 0f 10 11 13 ... 19 1a 1b ... 1f 20 21 23 ... 9f a0 a1 ... a9 aa ab ... b0 b1 ... ff.
Pvz.: #000000 - juoda; #ff0000 - raudona; ...

Trumpiniai: #000 - juoda; #f00 - raudona; #fff - balta. Trumpinti galima tik tada, kai abu iš dviejų spalvų verčių yra vienodos #ff0022 => #f02; #ffffff => #fff.

Permatomumas HEX sistemoje reguliuojamas pasitelkiant dar du papildomus skaitmenis (nesutrumpintame variante) HEX kodo gale. Permatomumas yra didžiausias esant 0 vertei, o mažiausias ff.

Positions

Static

Statinė pozicija pati nieko nedaro. Nei keičia elemento pozicija, nei suteikia jam galimybę naudotis top, left, right, bottom, z-index stiliais.
Statinė pozicija gali būti suteikiama kaip placeholder`is arba panaudota tam, kad nuimti kokią nors poziciją.

Relative

Relative pats iš savęs jokių elemento savybių nepakeičia. Tik pridėjus sukurtus nustatymus yra matomi pokyčiai. Nurodo ribas, kažkur jo viduje esančiam elementui su absolute pozicija.
Relative suteikia galimybę naudotis top, left, right, bottom, z-index stiliais.
Top, left, right, bottom, z-index nustatymai pastumia elementą nuo jo pradinės pozicijos, per nurodytą atstumą.

Absolute

Absoliutusis elementas išeina iš puslapio tėkmės. Nustatydamas savo poziciją jis naudoja pradinį view port`ą (matymo lauką). Suteikia galimybę naudoti Top, left, right, bottom, z-index nustatymus. Šie nustatymai elementą pastumia nuo pradinio ekrano vaizdo (view port`o) ARBA per nurodytą atstumą nuo nurodytos kraštinės (pradiniame view port`e) ARBA nuo artimiausio protėvinio elemento, kuris turi relatyviąją poziciją.

Fixed

Elementas su fixed pozicija išeina iš puslpaio tekmės. Suteikia galimybę naudotis top, left, right, bottom, z-index stiliais. Fiksuoja elementą vienoje viewport vietoje ir jis juda kartu su ekranu kai vartotojas scroll`ina per puslapį.
Top, left, bottom, z-index nustatymas nurodo nuo kurio matymo lauko (view port) vietos jis atsitraukia per nurodytą atstumą, nuo nurodytos kraštinės.
