  • grid-template-rows - sukuria grid tinklelio eilutes
  • grid-template - apjungia columns ir rows
  • gap, row-gap, column-gap - tarpas tarp grid`o elementų (grid-gap yra pasenusi sintaksė)
  • grid-template-areas - pavadinti tinklelio laukus
  • align-items -
  • justify-items -
  • place-items -
  • justify-content -

  • place-content - apjungia align-content ir justify-content
  • grid-auto-columns - sukuria papildomus stulpelius esant child elementams?
  • grid-auto-rows - sukuria papildomas eilutes esant child elementams?
  • grid-auto-flow - nurodo, kokia kryptimi grid`e išsidėsto elementai
  • grid - apjungia grid-template-rows, grid-template-columns, grid-template-areas,
  • grid-column-start - kuriame stulpelyje prasideda
  • grid-column-end - kuriame stulpelyje pasibaigia
  • grid-row-start - kurioje eilutėje prasideda
  • grid-row-end - kurioje eilutėje pasibaigia
  • grid-column - grid-column-start/grid-column-end
  • grid-row - grid-row-start/grid-row-end
  • grid-area - name | grid-row-start/grid-column-start/grid-row-end/grid-column-end
  • fraction - fr, paskirsto plotį tolygiai nurodytomis dalimis (alternatyva procentams)
  • repeat(3, 1fr) - sukuria 3 stulpelius/eilutes po 1 frakciją
  • minmax(min, max) - nurodo minimalų ir maksimalų stulpelio/eilutės plotį. repeat(3, minmax(10px, 1fr))
  • span - išplečia grid`o elementą per nurodytą laukelių skaičių (grid-column-end: span 3 - pasibaigia už trijų laukelių).
  • negative # - skaičiuoja stulpelį/eilutę nuo grid`o pabaigos.
